## 📓 About
----
- 픽쉐어는 여러 사람들과 함께 일상을 공유할 수 있는 다이어리 서비스입니다.

- 사진 뿐만 아니라 직접 그린 그림을 저장하여 일상을 업로드 할 수 있습니다.

- follow을 통해 사람들과 친구 관계를 맺고 댓글을 달며 소통할 수 있습니다.

- 나만 간직하고픈 일상은 lock 버튼을 통해 보호할 수 있습니다.
